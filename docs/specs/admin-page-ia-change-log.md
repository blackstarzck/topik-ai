# TOPIK AI Admin 페이지별 상세 IA 변경 로그

## 목적

- 페이지별 상세 IA 문서(`docs/specs/page-ia/*.md`)의 생성/수정 이력을 별도로 추적한다.

- 페이지별 상세 IA 문서를 생성하거나 수정하면 이 문서에 1건 이상 기록한다.
