# Admin MD Update Log

## 목적
- 코드/설정/플로우 변경으로 문서 수정이 발생했을 때, 변경 배경과 범위를 추적한다.

## 기록 규칙
- MD 문서를 수정한 작업마다 1건 이상 기록한다.
- 동일 요청에서 여러 MD를 수정했다면 한 항목으로 묶어 기록할 수 있다.
